# Ochena Bondhu (অচেনা বন্ধু)

Random video chat with strangers. Meet new people and make friends.

## Run Locally

```bash
npm install
npm start
```

- PC: `http://localhost:3000`
- Phone (same WiFi): `https://192.168.0.103:3443`

## Deploy on Vercel

Push to GitHub and import on [vercel.com](https://vercel.com).
